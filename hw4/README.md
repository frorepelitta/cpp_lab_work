
Money \
Добавление произвольного количества \
копеек (unsigned int) к объекту типа Money.\
Результат должен быть типа Money.\
Сделаем класс Money с двумя полями для копеек и рублей. \
У класса сделаем три конструктора, с двумя параметрами, одни, и без параметров. \
Далее создадим функцию для добавления к экземпляру класса Money копеек, функция правилно обрабатывает количество добавленных копеек, если копеек больше 100 то лишние копейки буду добавлены в рубли. \

Далее переопределим префиксный и постфиксный инкримен и декремент,сначала переопределим префиксный вариант и через него будем переопределять постфиксный.\
Далее переопределим приведение к целому числовому и дробному типу данных для экземпляров класса Money, целый числовой тип будет выводить рубли, дробный тип будет выводить копейки в формате рублей. \
Далее переопределим бинарные операции сложения и вычитания, добавить нечего.\
Еще переопредлим поток вывода так, чтобы при выводе эксземпляра класса на вывод поступали переменные рублей и копеек. \
